# 实现图片预览操作功能

## 1. 状态管理
在FileList组件中添加以下状态变量来跟踪图片操作：
- `scale`: 当前缩放比例（默认100%）
- `rotation`: 当前旋转角度（默认0度）

## 2. 核心功能实现
### 2.1 缩放功能
- 实现`handleZoomIn`函数：每次增加20%，上限300%
- 实现`handleZoomOut`函数：每次减少20%，下限20%
- 实现`handleResetScale`函数：重置缩放为100%

### 2.2 旋转功能
- 实现`handleRotateLeft`函数：向左旋转90度
- 实现`handleRotateRight`函数：向右旋转90度
- 实现`handleResetRotation`函数：重置旋转为0度

### 2.3 重置功能
- 实现`handleResetAll`函数：同时重置缩放和旋转

## 3. 界面更新
### 3.1 图片样式
修改图片元素，添加transform样式来实现缩放和旋转效果：
```css
transform: `scale(${scale / 100}) rotate(${rotation}deg)`
```

### 3.2 操作按钮
在Modal的footer中添加操作按钮组：
- 缩放控制：放大按钮、缩小按钮、重置缩放按钮
- 旋转控制：左转按钮、右转按钮、重置旋转按钮
- 整体重置按钮

### 3.3 状态显示
在图片预览区域添加当前状态显示：
- 显示当前缩放比例（如：100%）
- 显示当前旋转角度（如：0°）

## 4. 快捷键支持
添加键盘事件监听，支持以下快捷键：
- Ctrl++ 或 Ctrl+鼠标滚轮：放大
- Ctrl+- 或 Ctrl+鼠标滚轮：缩小
- Ctrl+←：向左旋转
- Ctrl+→：向右旋转

## 5. 性能优化
- 使用CSS transform进行缩放和旋转，利用GPU加速
- 确保状态更新时只重新渲染必要的组件
- 图片加载时使用占位符，避免空白闪烁

## 6. 交互设计
- 按钮使用清晰的图标和文字标识
- 按钮布局合理，易于点击
- 添加悬停效果，提升交互体验

## 7. 代码结构调整
- 将图片操作相关的状态和函数组织在一起
- 确保代码可读性和可维护性

## 8. 测试
- 测试所有操作按钮的功能
- 测试快捷键功能
- 测试边界情况（最大/最小缩放）
- 测试旋转360度的情况

通过以上实现，我们将为图片预览功能添加完整的操作支持，提供良好的用户体验。
